# Key Terms and Notions

- Query: The word the model is putting attention on right now
- Key: The values it is checking with
- Value: The final subject of the sentence

| Number       | Meaning |
|------------|-----|
| 1st      | The query value  | 
| 2nd        | The key value  | 


# How this works

1. It takes in a sentence in the beginning
2. It checks all the possibilites and pairs of query and keys (except itself)
3. Then it checks which one has the highest score out of all
4. Then it asks the user if it's subject selection is correct or not
5. If not, it adds the value to the query value of the word in it dictionary

> If the sentence is already given, it does not do anything.
